## 已知问题 & 坑

1. **MySQL 容器 ib_redo 损坏** — 86 上之前触发过,只能 `docker restart testmate-mysql` 重建(数据丢)。新部署务必在 .env 配大 `innodb_log_file_size` 避免频繁切换。
2. **86 没装 docker compose v2** — deploy.sh 已自动 fallback v1,但有些功能(如 `depends_on: service_healthy`)在 v1 行为略有差异,实测通过。
3. **后端 MySQL 密码含 `@`** — `TestMate@2026` 这种,DNS 解析要 `quote_plus`。已在 `app/core/config.py` patch 修。
4. **网络隔离** — 86 是 network_isolated,公网拉镜像失败,必须 offline-save 模式。
5. **monaco-editor 没用** — 装了没接入,bundle 多 1MB,AgentRunner 写完 prompt 后可拆 chunk 优化。
