- `/api` 走 Vite dev proxy,生产走 Nginx 反代

- `scripts/dev.sh`:Mac 本地起(分开起后端 / 前端,不依赖 docker)
- `scripts/stop.sh`:关掉本地进程

## 端到端 demo 流

```
[用户] 浏览器打开 http://localhost:8080
  → 登录(admin / admin123,P0 手动建用户)
  → 进入 /diagnosis
  → 点「填入示例 log」(或自己贴)
  → 点「开始诊断」
    → 前端 POST /api/v1/diagnose/stream (JWT 鉴权)
    → 中台截断 log → 转发 Dify /chat-messages (SSE)
    → 中台边收边 yield,前端边读边渲染
  → 右侧 Monaco 风黑底色面板流式显示 AI 结论
```
